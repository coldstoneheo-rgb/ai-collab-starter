# Compliance & Legal

This directory contains compliance requirements, legal constraints, and regulatory documentation.

## Contents

- Legal requirements (GDPR, COPPA, etc.)
- Privacy policies
- Security standards
- Regulatory compliance
- Terms of service
- Data retention policies

## Maintenance

- Review quarterly or when regulations change
- Changes require legal team approval
- Document all regulatory changes
- Keep audit trail

## Important

⚠️ **All changes to this directory must be reviewed by legal counsel**

## Examples

```
compliance/
  privacy/
    gdpr_compliance.md
    data_protection.md
  security/
    security_standards.md
    incident_response.md
  legal/
    terms_of_service.md
    privacy_policy.md
```
