# ai-collab-starter

Starter template to integrate multiple AI agents (Claude, Gemini, Perplexity, GPT) into GitHub-based dev workflows.

## Quick start

1. Copy this template into a new repo.
2. Set secrets in GitHub repository settings:
   - CLAUDE_API_KEY
   - GEMINI_API_KEY
   - PERPLEXITY_API_KEY
   - GPT_API_KEY
3. Run:
   ```bash
   ./scripts/init_project.sh
